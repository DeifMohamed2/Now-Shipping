## 5. FEES AND PAYMENT TERMS

### 5.1. Fee Structure

**Delivery Fees:**
Delivery fees are calculated based on:
- Destination government/region (Cairo, Alexandria, Delta-Canal, Upper-RedSea)
- Order type (Deliver, Return, Exchange, Cash Collection)
- Express shipping option (if selected)

Base fees by region:
- **Cairo:** Deliver (80 EGP), Return (70 EGP), Cash Collection (70 EGP), Exchange (95 EGP)
- **Alexandria:** Deliver (85 EGP), Return (75 EGP), Cash Collection (75 EGP), Exchange (100 EGP)
- **Delta-Canal:** Deliver (91 EGP), Return (81 EGP), Cash Collection (81 EGP), Exchange (106 EGP)
- **Upper-RedSea:** Deliver (116 EGP), Return (106 EGP), Cash Collection (106 EGP), Exchange (131 EGP)

Express shipping fees are calculated as 2x the base fee for the selected order type.

**Return Fees:**
Return fees are calculated based on:
- Base return fee (15 EGP for standard returns, 20 EGP for exchanges)
- Return condition (excellent, good, fair, poor, damaged) - affects fee multiplier
- Express shipping option (1.5x multiplier if selected)
- Destination government (Cairo: 1.0x, Alexandria: 1.2x, Other: 1.5x)
- Processing fee (10% of calculated fee)
- Inspection fee (5 EGP for return orders)

**Pickup Fees:**
Pickup fees are calculated based on:
- Destination government/region
- Number of orders picked up (discount for 3+ orders)

Base pickup fees:
- **Cairo:** 50 EGP (65 EGP if < 3 orders)
- **Alexandria:** 55 EGP (72 EGP if < 3 orders)
- **Delta-Canal:** 60 EGP (78 EGP if < 3 orders)
- **Upper-RedSea:** 80 EGP (104 EGP if < 3 orders)

### 5.2. Payment Methods

### 5.3. Cash Cycles and Settlements

**Cash Cycle Process:**
- Completed orders are processed in weekly cash cycles
- Money release dates are calculated based on order completion dates
- Funds are released on Wednesdays following the completion date
- If an order is completed on a Wednesday, funds are released the following Wednesday
- If an order is completed on any other day, funds are released on the next Wednesday

**Settlement Status:**
- **Pending:** Transaction created but not yet included in a release
- **Included in Release:** Transaction included in an upcoming release batch
- **Settled:** Transaction has been processed and funds released

**Transaction Types:**
- Cash cycle transactions (order completions)
- Fee transactions (delivery, return, pickup, express fees)
- Refund transactions
- Deposit and withdrawal transactions
- Shop order delivery transactions
